Frequently Asked Questions

How can I register in XFiles?
To register for Xfiles click the green button "Register" (button at the top right) on the main page of the site. The dialog appears, where you should enter your e-mail, which will be your user name in our system, then enter your password, then repeat your password, and after that enter your name. If you like you may enter more information.
Note that e-mail, password, first name and last name are mandatory parameters. You will not be able to complete registration if you don't fill in this fields.
How to sign in to XFiles?
To log in to your account, click the green button "Sign In" (the button is located on the top right), enter your e-mail address (user name) and password.
I forgot the password to my account and cannot sign in. What should I do?
You can reset the password with the help of password recovery page. The password will be sent to you e-mail.
What are the buttons in the main menu?
On the top of the any page you can see the main menu. It's just a set of links that helps to navigate between pages of the site.
Currently, it contains the following: Home (return to home page), About (about us and our file storage), Services, Blog (still in development), Pricing (in development), Cabinet (your personal account management, where you can also register, log in to your account and reset your password and log out).
How to upload files?
You can easily upload your file to XFiles. Sign in to your account and navigate to "Files" tab in personal cabinet. You will see the "Upload". By clicking this button you will be navigated to the "File upload form". Just select file to upload, specify necessary parameters and click "Upload".
Is it possible to upload several files at a time?
Unfortunately, we have not implemented this feature yet. But you can upload an archive of files (ZIP, RAR and others).
What is the maximum size of uploaded file?
We are at the stage of testing. So, the maximus size of uploaded file is currently set to 40 Mb.
What is the File Access Type?
There are 3 access types of files: Private Access, Public Access and Group Access.
Files with "Public Access" type can be downloaded by any user of XFiles who has the download-link to the file (of course, the user will be able download it if he knows the password).
Files with "Private Access" type can be downloaded by the owner of the file. Currently, the file can only have a single owner.
"Group Access" files (or just group files) are shared between a group of users. But in order to download the file, the user will need to get approval of at least 3 other users of the group.
How to upload file and give a link to my friend?
If you want to share the file with your friends, just upload it with "Public" access type. You can give the download-link to another user.
If you want to send file to a person who is not registered in our service, you can create a download request and, after file is read, send the link to ready file to your friend.
I downloaded a lot of files but all my requests disappeared. Why?
Since the download request is in READY status (2), it exists only 2 hours. If you want to download the file again, just create another request.
How can I delete, add or edit my files?
Currently, you are not able to do that. We are working on it.
What can I do in Messages tab?
Initially, this tab was intended for messaging between users. But this feature is not very important.
Now, Messages tab is used for approving download requests for group files.
