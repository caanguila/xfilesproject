About XFiles

Welcome to our file hosting «XFiles»! We are proud to announce that our project have recently started, and our services are being actively developed.

Unlike others, our XFiles can offer you, in addition to a simple file storage, a file storage with encryption. It is also possible to protect the file from unwanted download with 3-user approval. This means that the uploaded file can be downloaded only if the download request received confirmation of three members of the group. It obviously improves the the file storage making it many times more secure.

At current alpha-testing stage of development, XFiles is a simple file hosting, so many things are not ready yet. You can take part in development, to help us to improve the project by finding bugs and errors. If you want to see a feature in our project which is not implemented, feel free to send us a feature request.

Also, soon we will create a section where you will be able to share with us your ideas and ask questions.

Thank you very much four your patience.
